# 숫자 야구 게임

## 요구사항
* 상대방 역할은 컴퓨터이며, 1~9 중 서로 다른 임의의 수 3개를 선택한다.
* 사용자에게 1~9까지 서로 다른 수로 이루어진 3자리 수를 입력 받는다.
* 같은 수가 같은 자리에 있으면 "스트라이크" 출력한다.
* 같은 수가 다른 자리에 있으면 "볼" 출력한다.
* 같은 수가 없으면 "포볼" 또는 "낫싱" 출력한다.
* 위의 출력 힌트를 보며 컴퓨터의 3개의 숫자를 모두 맞히면 게임 종료한다.
* 사용자가 잘못된 값을 입력하면 [ERROR]로 시작하는 에러메세지 출력하며 게임은 계속 진행한다.

## 구현할 기능 목록
* 1~9 범위의 서로 다른 임의의 수 3개 추출
* 사용자의 입력값 받기
* 사용자 입력값이 1~9 범위의 서로 다른 수인지 확인 (문자가 포함되었는지, 자리수가 초과했는지, 중복된 수가 있는지 등)
* [ERROR] 메세지 출력
* 추출한 수와 입력받은 수의 숫자 비교
* 비교한 값에 따른 출력 힌트 출력
* 게임 종료
