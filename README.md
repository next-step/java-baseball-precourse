# 숫자 야구 게임
## 진행 방법
* 숫자 야구 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 과제 제출 과정
* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/precourse)

## 구현 할 기능 목록
* 상대방(컴퓨터)의 랜덤 숫자 생성
    - 3자리의 랜덤 숫자 생성 함수
    - 생성된 3자리의 랜덤 숫자가 서로 다른 수로 이루어진 3자리의 수인지 체크하는 함수
    - 각 자리별 숫자를 가져오는 함수
* 사용자의 숫자를 입력받는 함수
    - 입력된 항목이 3자리의 숫자가 맞는지 체크하는 함수
* 입력된 사용자의 숫자와 상대방(컴퓨터)의 랜던 숫자를 비교하는 함수
    - 숫자 비교 후 같은지 다른지 확인하는 함수
    - 각 자리별 숫자 중 같은 숫자가 있는지 확인하는 함수
    - 다른 자리에 해당하는 숫자 중 같은 숫자가 있는지 확인하는 함수
* 입력된 사용자의 숫자에 대한 결과값을 출력하는 함수
* 게임을 계속 진행 할 지 종료 할 지 확인하는 함수
