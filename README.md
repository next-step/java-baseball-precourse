# 숫자 야구 게임
## 진행 방법
* 숫자 야구 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 야구게임 프로세스
1. 야구게임을 시작을 위한 1~9까지 서로 다른 수로 이루어진 3자리의 수를 가진 Computer를 생성한다.
2. Gamer는 1~9까지 서로 다른 수로 이루어진 3자리의 수를 입력한다.
3. 게임은 Role에 맞게 Gamer가 입력한 값과 Computer가 가진 수를 비교하여 스트라이크와 볼 등을 판정한다. 
	- Computer가 가진 수의 자릿수와 값이 Gamer가 입력한 수의 자릿수와 값이 같으면 : n 스트라이크
	- Gamer가 입력한 각 자릿수별 값이 Computer가 가진 각 자릿수별 값과 일치하지 않지만 다른 자릿수에 값이 존재할 때 : n 볼
4. 판정 된 스트라이크와 볼 수로 게임 결과를 출력한다.
	- 스트라이크와 볼 존재 : n 스트라이크 n 볼 출력
	- 스트라이크만 존재 : n 스트라이크 출력
	- 볼만 존재 : n 볼 출력
	- 스트라이크와 볼 모두 없을 시 : 낫싱 출력
	- 3 스트라이크 시 : 게임 승리
5. 3 스트라이크 후 게임 승리 시 새로운 Computer의 수로 재시작할 지 종료할지 선택한다.

* 입력 값 체크) 숫자 입력 및 재시작, 종료 선택 시 null, 비숫자, 자릿수 등의 값은 ERROR 처리하여 재입력 하도록 한다.
