# 숫자 야구 게임

## 요구사항 정의

### 1. 랜덤숫자 생성
- [ ] 1부터 9까지의 숫자 생성 

### 2. 입력 값의 유효성 검사 
- [ ] 세 자리수인지
- [ ] 각 자리수의 값이 1~9 범위내에 속하는지
- [ ] 각 자리수의 값이 중복이 없는지

### 3. 숫자 비교 및 결과 반환
- [ ] **strike** : 두 수의 값과 위치가 같을 때
- [ ] **ball** : 두 수의 값만 같을 때
- [ ] **nothing** : 값과 위치가 모두 다를 때

### 4. 분기 로직
- [ ] 매 회마다 사용자로부터 숫자를 입력받음
- [ ] 입력받은 숫자 유효성 검사
  - [ ] 유효성 검사 실패시, 사용자에게 재입력 요청
- [ ] 숫자 비교
   - [ ] 정답일 경우, 사용자에게 새 게임 시작 여부를 질의 (1 : 새 게임 시작, 2 : 게임 종료)
   - [ ] 오답일 경우, ``숫자 비교 결과 메시지`` 출력 후 사용자에게 숫자 재입력 요청

### 5. 화면
- [ ] 매 회 마다 ``"숫자를 입력해주세요"`` 출력
- [ ] 입력 값의 유효성 검사 결과에 따라 메시지 출력
    - 유효성 검사 실패시, ``"1부터 9까지의 숫자를 입력하세요"`` 혹은 ``"세 자리수를 입력하세요"`` 메시지 출력 후, 사용자에게 숫자 재입력 요청
- [ ] 숫자 비교 결과에 따라 메시지 출력
    - 정답일 경우, ``"3개의 숫자를 모두 맞히셨습니다! 게임종료"``, ``"게임을 새로 시작하려면 1, 종료하려면 2를 입력하세요"`` 출력 후, 사용자에게 입력 요청
    - 오답일 경우, ``"x스트라이크"`` 혹은 ``"x볼"`` 출력 후, 사용자에게 숫자 재입력 요청


## 진행 방법
* 숫자 야구 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.
