# 숫자 야구 게임
## 진행 방법
* 숫자 야구 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 과제 제출 과정
* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/precourse)

## 구현 기능 목록
1. 입력 값에 따라 반복해서 게임을 진행하는 기능.
2. 게임 시작 시 랜덤한 숫자 저장(1~9까지 서로 다른 수로 이루어진 3자리 수)
3. 입력받은 숫자에 따라 동일한 숫자면 게임 종료
4. 입력값에 따라 스트라이크/볼 구분해 출력
5. 도메인 로직 단위테스트 구현
