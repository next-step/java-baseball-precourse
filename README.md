# 숫자 야구 게임
## 진행 방법
* 숫자 야구 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 기능요구사항
* 기본적으로 1~9 사이 서로 다른 수로 이루어진 3자리수를 맞추는 게임
* 컴퓨터는 임의에 다른 3자리 임의숫자른 입력
* 같은수가 같은자리에 있으면 스트라이크, 다른자리에 있으면 볼, 같은수가 없으면 낫싱을 이용해 컴퓨터에 수를 맞추면 능리
* 플레이어는 3자리숫자를 입력하여 위에 과정을 반복해 숫자를 모두 맞치면 종료
* 게임을 종료된 후 게임을 다시시작하거나 종료할수 있음

## 기능목록
* 컴퓨터의 3자리 임의값 설정
* 플레이어에 입력값 검증처리 및 변경처리
* 플레이어에 입력값에 따른 결과값 출력처리
* 게임이 끝난 후 재시작 또는 종료 처리 1
