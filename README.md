# 숫자 야구 게임
## 진행 방법
* 숫자 야구 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 과제 제출 과정
* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/precourse)

## 기능 요구사항 정리
### 플레이어 
* 플레이어는 1부터 9까지 서로 다른 수로 이루어진 3자리 수를 입력한다.
    * 중복된 숫자가 입력되거나 3자리로 이루어진 수가 아닌경우 ERROR

### 출제자 (컴퓨터)
* 출제자는 1부터 9까지 서로 다른 수로 이루어진 3자리 수를 정답으로 정한다.
* 정답은 랜덤하게 결정된다.

### 확인자
* 확인자는 플레이어가 입력한 3자리와 컴퓨터가 설정한 정답 3자리를 비교한다
* 비교 결과 같은 수가 같은자리에 있으면 스트라이크, 다른 자리에 있으면 볼, 같은 수가 전혀없으면 포볼 또는 낫싱이란 힌트를 제공한다.

### 게임
* 게임은 플레이어가 상대방의 숫자를 맞추면 종료한다.
* 게임이 종료 될 때까지 플레이어에게 입력값을 받는다
* 게임을 종료한 후에 게임을 다시 시작하거나 완전히 종료할 수 있다.
* 사용자가 잘못 입력한 경우 [ERROR]로 시작하는 에러 메세지를 출력하고 게임을 계속 진행할 수 있어야 한다.

### 공통 
* 중복되지 않는 3자리수의 숫자를 보장해주는 기능
* 중복되거나 3자리가 아닌 숫자를 받으면 Exception을 발생
