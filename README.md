# 숫자 야구 게임
## 진행 방법
* 숫자 야구 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 과제 제출 과정
* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/precourse)

## 기능 목록
1. 컴퓨터의 숫자를 생성한다
   * 1 ~ 9의 값을 가지는 숫자 3개를 생성한다
   * 같은 숫자를 가질 수 없다
   
   
2. 사용자의 입력을 받는다
   * 숫자만 입력을 받는다.
   * 잘못된 입력을 할 경우 IllegalArgumentException을 발생시키고 게임을 종료한다
   

3. 컴퓨터의 숫자, 사용자의 숫자를 비교한다
   * 위치와 숫자가 같으면 스트라이크, 위치는 다르지만 숫자를 포함하면 볼
   

4. 비교 결과를 바탕으로 메시지를 출력한다
   * 3스트라이크인 경우: 3개의 숫자를 모두 맞히셨습니다! 게임 종료
   * 스트라이크와 볼이 섞인 경우: x볼 y스트라이크
   * 스트라이크와 볼이 없는 경우: 낫싱
   

5. 비교 결과를 바탕으로 다음 작업을 진행한다
   * 3스트라이크인 경우
     * 게임 재시작 질문 메시지를 출력
       * 메시지: 게임을 새로 시작하려면 1, 종료하려면 2를 입력하세요.
     * 사용자의 입력을 받음
       * 1인 경우: 1번 기능부터 다시 시작
       * 2인 경우: 게임 종료
       * 잘못된 입력을 할 경우 IllegalArgumentException을 발생시키고 게임을 종료한다
   * 그 외의 경우: 2번 기능부터 다시 시작
