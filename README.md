# 숫자 야구 게임

## 구현할 기능 목록

- 관련 객체
    - Main, UI, BaseballGame(컴퓨터), User

1. 전체 프로그램 sequence 관리 - Main
2. baseball Game 정답 숫자 생성하기 - Game
3. 유저에게 숫자 입력 요청문 출력하기 - UI
4. 숫자 입력하기 - User(Console)
5. 유저 입력 validation 하기 - Game
6. 유저 입력 채점하기 (스트라이크, 볼, 낫싱) - Game
7. 유저가 맞췄는지 여부 알아내기 - Game
8. 유저가 모두 맞춰 게임이 끝났는지 여부 판단 - Game
9. 게임 Grade 초기화 하기 - Game
10. 게임 종료 메시지 출력하기 - UI
11. 게임 재시작 질의 출력하기 - UI
12. 게임 재시작여부 질의 판단하기 - UI
