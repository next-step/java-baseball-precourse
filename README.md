# 숫자 야구 게임
## 진행 방법
* 숫자 야구 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 과제 제출 과정
* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/precourse)

## 기능
* 1부터 9까지 서로 다른 수로 이루어진 3자리의 수를 무작위로 생성한다.
* 사용자로부터 3자리의 수를 입력받아 무작위로 생성된 숫자 3개를 맞히면 게임은 종료된다.
* 같은 숫자가 같은 자리에 있으면 스트라이크, 다른 자리에 있으면 볼, 같은 숫자가 없으면 낫싱을 출력한다.
  * e.g.
    * 2볼 1스트라이크
    * 3스트라이크
    * 낫싱
* 게임이 종료되면 새 게임을 시작하거나 종료할 수 있다.
* 잘못된 값이 입력되면 IllegalArgumentException을 발생하고 종료된다.
