# 숫자 야구 게임
## 진행 방법
* 숫자 야구 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 과제 제출 과정
* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/precourse)

## 구현할 기능 목록
* 상대방(컴퓨터) 난수 생성
    * 1 ~ 9의 숫자로만 이루어진 제한된 자릿수의 문자열
    * 중복되는 숫자의 문자가 존재하지 않는 문자열
    
* 사용자 입력
    * 1 ~ 9의 숫자로만 이루어진 제한된 자릿수의 문자열
    * 중복되는 숫자의 문자가 존재하지 않는 문자열
    
* 점수 계산
    * 같은 숫자의 문자이면서 같은 위치 : 스트라이크
    * 같은 숫자의 문자이면서 다른 위치 : 볼
    * 같은 숫자의 문자 없음 : 낫싱
    
* 계임 순서
    * 상대방(컴퓨터) 난수 생성
    * 게임 세팅 초기화
    * 사용자 입력
    * 라운드 진행 or 종료 
    * 게임 진행 or 종료

* 주의 사항
    * 점수 계산 시
        * 라운드 종료 조건에 맞지 않다면 상대방(컴퓨터) 난수 생성 하지 않는다.
        * 매 라운드 시작 시 Strike, Ball 초기화
