# 숫자 야구 게임
## 진행 방법
* 숫자 야구 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 과제 제출 과정
* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/precourse)

## 구현할 기능 목록
### 1) 게임볼 생성
* 게임볼 숫자 생성시 1부터 9까지의 수로 이루어진 3자리 수를 생성
* 컴퓨터 게임볼 숫자는 랜덤으로 생성
* 사용자입력 게임볼 숫자는 입력된 3자리 값으로 생성
* 게임볼은 서로 다른 중복되지 않는 값으로 생성


### 2) 생성된 게임볼로 게임 플레이
* 게임 룰 구현
1. 동일 숫자 && 동일 위치 = 스트라이크
2. 동일 숫자 && 다른 위치 = 볼
3. 동일 숫자 없음 = 나싱(nothing)
4. 모든 숫자 동일 (값,위치) = 스트라이크 아웃 (게임승리)

* 게임 결과 표시
1. 게임결과 저장
2. 게임결과 출력


### 3) 게임볼 생성시 예외처리
* 3자리가 아닌 경우
* 숫자가 아닌 경우
* 중복 숫자인 경우

### 4) 콘솔 입력
* 사용자 숫자 입력 (정답을 맞출 때까지 입력처리)
* 사용자 새게임 여부 입력 (1 : 새게임, 2: 게임종료)
