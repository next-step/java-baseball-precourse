# 숫자 야구 게임
## 진행 방법
* 숫자 야구 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 과제 제출 과정
* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/precourse)

## 구현 목록
* 게임 시작 시 초기화로직 요청 함수 생성
* 데이터 초기화 로직 구현
  * 모델의 숫자들을 모두 랜덤한 숫자로 초기화
* 게임 진행로직 요청 함수 생성
* 상대방에게 숫자입력 요청하고 입력데이터 받는 함수 구현
  * 요청 문구 출력
  * 입력데이터가 요청한 데이터가 맞는지 유효성 검증
* 입력데이터와 랜덤숫자가 맞는지 비교하는 함수 구현
* 결과값에 따라 숫자입력을 받을지 종료문구를 받을지 판단하는 함수 구현
* 종료시 출력문구와 재시작여부 입력 요청 함수 구현
* 재시작여부 입력데이터가 보기 중 주어진 숫자가 맞는지 유효성 검증
* 재시작 입력 시 다시 게임시작 요청함수 호출 구현
