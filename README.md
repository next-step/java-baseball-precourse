# 숫자 야구 게임
## 진행 방법
* 숫자 야구 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 과제 제출 과정
* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/precourse)

## 구현 기능 목록
1. 컴퓨터는 1~9 사이의 랜덤한 숫자 3가지를 선택한다. 
2. 숫자를 모두 맞출 때까지 게임을 실행한다. 
3. 플레이어가 입력한 숫자와 컴퓨터가 선택한 숫자를 비교하여 결과를 리턴한다. 
4. 게임을 계속할지 그만둘지 결정한다.
5. 사용자의 입력값에 대한 유효성을 체크한다. (숫자 입력, 게임 종료 여부 입력)
