# 숫자 야구 게임
## 진행 방법
* 숫자 야구 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 과제 제출 과정
* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/precourse)


---
## 구현할 기능 목록
1. 시스템 메세지 정의
   - 1.1. 숫자 비교 힌트 메세지
   - 1.2. 정답 메세지
   - 1.3. 예외 메세지


2. 3자리 숫자 랜덤 생성 기능


3. 사용자 Console 입력 기능
   - 3.1. 숫자가 아닌 문자 입력 시 예외 처리
   - 3.2. 3자리수 입력이 아닐 시 예외 처리


4. 사용자 입력 숫자 비교 기능
   - 4.1. 스트라이크, 볼, 포볼 카운트 
   - 4.2. 틀릴 시 반복
   - 4.3. 정답 처리
