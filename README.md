# 숫자 야구 게임
## 진행 방법
* 숫자 야구 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 과제 제출 과정
* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/precourse)

## Features

- 새 게임이 시작되면 프로그램은 1~9 사이의 중복되지 않는 3자리 숫자를 생성한다
- 사용자가 3자리 숫자를 입력하면 프로그램은 앞서 생성한 숫자와 비교해서 스트라이크, 볼 등을 출력한다
  - 사용자가 3자리 숫자를 모두 맞출 때까지 반복한다
  - 만약 사용자 입력이 잘못된 형식이면 '[ERROR]'로 시작하는 메시지를 출력한다
- 사용자가 3자리 숫자를 모두 맞추면 축하 메시지를 출력한다
- 사용자가 3자리 숫자를 모두 맞추면 새로운 게임을 시작하거나 프로그램을 종료할 수 있다
