# 숫자 야구 게임

---------
## 진행 방법
* 숫자 야구 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 과제 제출 과정
* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/precourse)

## 기능 구현
### 1. 야구번호 생성
* 기능
    * 랜덤한 숫자를 생성함

* 조건

    * 1과 9 사이에 3자리의 숫자를 생성
    * 숫자는 중복 불가능하다

### 2. 야구번호 입력
* 기능
    * `사용자`가 직접 숫자를 입력
* 조건
    * 숫자는 총 3개 입력
    * 숫자는 서로 중복되면 안됨
    * 문자나 띄어쓰기가 적용되면 안된다
    * 잘못된 값을 입력하면 [Error] 메시지를 출력하고 반복

### 3. 야구번호 검증

* 기능
    * `사용자` 가 입력한 숫자와 랜덤숫자를 비교하여 결과를 출력
* 조건
    * 메시지는 숫자 야구게임 규칙 __(쓰트라이크, 볼, 나싱)__ 을 출력
