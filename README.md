# 숫자 야구 게임
## 진행 방법
* 숫자 야구 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 과제 제출 과정
* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/precourse)

## 구현 기능 목록
* 플레이어 Model 개발 - 입력받은 숫자 저장 기능
* 컴퓨터 Model 개발 - 랜덤 숫자 생성 기능
* 게임 Model 개발 - 게임 진행, 입력된 숫자 비교 기능
* 게임 View 개발 - 사용자에게 메세지 출력, 메세지 입력받는 기능
* 게임 Controller 개발 - 게임 시작, View로부터 입력된 숫자를 바탕으로 게임 결과 산출
* 입력값 검증 기능 개발
