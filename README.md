# 숫자 야구 게임
## ☄기능 목록

1. 게임 시작시, 시스템은 플레이어가 맞출 3자리의 랜덤 숫자를 생성
2. 플레이어는 3자리 숫자를 입력
   - 숫자가 아닌 타입의 값과 3자리가 아닌 값을 입력받을 경우, 아래 에러 메시지를 출력하고 게임을 계속 진행한다.
     - "[ERROR] 숫자가 아닙니다. 다시 입력하세요."
     - "[ERROR] 입력값에 0이 있습니다. 다시 입력하세요."
3. 플레이어가 입력한 3자리 숫자와 시스템이 생성한 숫자를 비교
   - 같은 수가 같은 자리에 있으면 "(해당하는 개수)스트라이크"를 출력한다.(ex) 2스트라이크)
   - 같은 수가 다른 자리에 있으면 "(해당하는 개수)볼"을 출력한다.(ex) 3볼)
   - 같은 수가 전혀 없으면 "낫싱"을 출력한다.
   - 시스템이 생성한 숫자과 플레이어가 입력한 숫자가 일치하는 경우 아래 메시지를 출력한다.
     - "3스트라이크\</br> 3개의 숫자를 모두 맞히셨습니다! 게임 끝"
4. 게임이 종료되면 아래 메시지를 출력하고 플레이어는 게임을 다시 시작하거나 완전히 종료할 수 있다.
   - "게임을 새로 시작한다면 1, 종료하려면 2를 입력하세요"



## 🔎진행 방법
