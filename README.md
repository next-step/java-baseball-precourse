# 숫자 야구 게임
## 진행 방법
* 숫자 야구 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 과제 제출 과정
* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/precourse)

- 루프 시작
- 입력
- 예외처리
- 랜덤숫자 생성
- 조건분기
  - 같은 숫자 포함, 위치 동일 : 스트라이크
  - 같은 숫자 포함, 위치 다름 : 볼
  - 같은 숫자 없음 : 낫싱
- 매 입력마다 위 결과 출력    
- 모든 숫자 및 위치 동일하면 게임 종료
- 메인 종료 / 게임 재시작 분기
