# 숫자 야구 게임
## 진행 방법
* 숫자 야구 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 과제 제출 과정
* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/precourse)

## 과제 기능 목록

* 서로 다른 3 자리수를 랜덤하게 생성
* 게임 사용자로부터 숫자 3개를 입력값으로 받음
* 게임 사용자로부터 받은 입력값과 생성됨 값을 비교하여 게임 결과 출력
* 특정 게임 조건까지 게임 반복 기능 추가
* 테스트 케이스 완성
