# java-baseball-precourse

## 1. Application

- 실제 프로그램의 시작. main 함수
- 컴퓨터의 숫자 생성
- 사용자의 숫자를 받아들임, 오류 검출
- 사용자의 숫자와 컴퓨터의 숫자 비교
- 비교 결과 출력
- 3스트라이크가 아닐 경우 게임 계속 진행
- 3스트라이크일 경우 게임 재시작 여부를 묻고 게임 종료 또는 재시작.


## 2. Error

- 사용자가 입력한 숫자의 오류를 찾아냄
- 중복된 숫자(112, 223 등…)
- 3자리가 아닌 숫자(1234, 3553 등…)
- 숫자가 아닌 문자열(lion, ??, 안녕 등…)


## 3. GameState

- 게임의 진행, 게임의 결과 출력, 게임의 재시작 여부를 다룬다.
- 게임 진행 시 “숫자를 입력해 주세요 : ” 문구를 출력하고 사용자가 입력한 숫자를 받아들인다.
- 사용자가 입력한 숫자를 토대로 게임의 결과를 출력한다.
    - 1ball 1strike, nothing…
- 게임의 재시작 여부를 확인한다.
    - 3strike시 사용자의 입력을 받아 게임을 재시작 할 지 종료할 지 결정.


## 4. Utiltools

- 잡다한 프로그램에 실행 시 필요한 기능들을 모아 놓았다.
- 상대방(컴퓨터)의 랜덤한 3자리 숫자 생성
- 문자열을 숫자 배열로 변경
- 두 숫자를 비교
- 정수 배열을 0으로 초기화
