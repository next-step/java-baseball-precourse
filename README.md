# 숫자 야구 게임
## 진행 방법
* 숫자 야구 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 과제 제출 과정
* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/precourse)

## 기능
- 세 자리 숫자를 생성한다.
  - 세자리 숫자는 각기 다른 값이어야 한다.
- 사용자가 입력한 숫자와 생성한 숫자를 비교한다.
  - 숫자의 자리와 값이 같은 경우 스트라이크를 반환
  - 숫자가 3자리 중에 있으면 볼을 반환
- 3스트라이크가 나온 경우 게임을 종료할지 새 게임을 진행할지 결정한다.
