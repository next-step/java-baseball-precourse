# 숫자 야구 게임
## 정의 (definition)
* 1부터 9까지 서로 다른 수로 이루어진 3자의 수를 맞추는 게임
* 같은 수가 같은 자리에 있으면 스트라이크, 다른 자리에 있으면 볼, 같은 수가 전혀 없으면 포볼 또는 낫싱이라는 힌트를 얻고, 그힌를 이용해 먼저 상대방의 수를 맞추면 승리한다.
* 상대방(컴퓨터)는 1에서 9까지의 서로 다른 임의의 수 3개를 선택한다.
* 게임 플레이어는 3개의 숫자를 입력하고 컴퓨터는 입력한 숫자에 대한 결과를 출력한다.
* 이 과정을 반복해 플레이어가 컴퓨가 선택한 3개의 숫자를 맞추면 승리한다.
* 게임을 종료한 후 게임을 다시 시작하거나 완전히 종류할 수 있다.

## 기능 목록 (function list)
* 컴퓨터는 서로 다른 세개의 숫자를 선택한다. (computer chooses three different numbers (1-9))
* 플레이어는 3개의 숫자를 입력한다. (input three different numbers each 1-9)
* 플레이어 올바른 숫자를 입력했는지 검증한다.
* 잘못입력했다면, 에러메시지를 출력한다.
* 플레이가 입력한 숫자와 컴퓨터가 선택한 숫자에 대한 스트라이크, 볼 갯수를 리턴한다.(return strikes and ball numbers)
* 스트라이크, 볼 갯수에 따라 힌트를 출력한다.(print hint)
* 차이가 없다면 과정을 종료한다. (close procedures)
