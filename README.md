# 숫자 야구 게임

## 기능 요구사항 📑
* 컴퓨터는 서로 다른 숫자 3개를 선택 할 수 있다.
  *[ ] 랜덤함수를 이용하여 숫자 3개를 선택한다.
  *[ ] 숫자의 범위가 1~9 사이이다.
  *[ ] 숫자에 중복이 있는 경우가 생기면 안된다.

    
* 게임 결과를 규칙에 따라 판정할 수 있다.
  *[ ] 자리가 같은지 비교 한다.
  *[ ] 숫자가 같은지 비교 한다.
  * 같은 숫자가 같은 자리를 가지고 있다면 스트라이크 이다.
  * 숫자는 같지만 자리가 다르면 볼이다.
  * 같은 숫자가 없다면 낫싱이다.
  
  
* 사용자가 잘못된 값을 입력할 경우 게임을 계속 진행할 수 있어야 한다.
    *[ ] 문자열 형태의 숫자를 숫자 리스트로 변환한다.
    *[ ] 숫자의 범위가 1~9 사이이다.
    *[ ] 숫자에 중복이 있는 경우가 생기면 안된다.
    *[ ] 입력된 숫자는 3자리 여야 한다.
    *[ ] 예외 상황 발생시 throw한다.
      *[ ] [ERROR]로 시작하는 에러 메시지를 출력한다.
  

* 게임을 다시 시작하거나 완전히 종료할 수 있다.
* 게임 종료 조건을 판단할 수 있다.
