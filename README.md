# 숫자 야구 게임

## 진행 방법

* 숫자 야구 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 과제 제출 과정

* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/precourse)

## 기능 목록

* 게임시작시 컴퓨터는 1-9까지의 서로 다른 수로 이루어진 3자리의 숫자를 생성한다.
* 플레이어는 컴퓨터의 숫자를 맞추기 위한 3자리 숫자를 입력할 수 있다.
    - 입력시 3자리 숫자가 아닌 경우 [ERROR]로 시작하는 에러메세지를 출력 후 게임을 진행한다.
* 플레이어가 입력한 숫자의 결과(힌트)를 출력한다.
    - 힌트 리스트
        + 입력한 숫자의 위치와 숫자가 맞을 경우 스트라이크
        + 입력한 숫자의 위치는 다르지만 숫자가 맞을 경우 볼
        + 아무것도 맞추지 못한경우 낫싱
* 플레이어가 3개의 숫자와 위치를 모두 맞춘 경우 게임을 종료한다.
    - 종료 메세지("3개의 숫자를 모두 맞히셨습니다! 게임 끝") 출력
* 게임 종료시 게임을 다시 시작하거나 완전히 종료할 수 있다.
    - 메뉴("게임을 새로 시작하려면 1, 종료하려면 2를 입력하세요.") 출력
    - 입력시 메뉴가 아닌 숫자를 입력할 경우 [ERROR]로 시작하는 에러메세지를 출력 후 메뉴를 다시 출력한다.
