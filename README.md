### Model
#### Computer 
    - 컴퓨터의 난수 생성을 위한 클래스
    - 같은 수 중복 여부 확인 
#### Score 
    - 사용자의 추리에 따른 스트라이크, 볼 개수 계산 

### Utils
#### ValidateAnswer 
    - 볼과 스트라이크의 개수를 판단 
#### ValidateRestart
    - 재시작 여부 판단 

### View
- 사용자의 입력 여부에 따라 inputView, OutputView로 분리 
#### inputView
    - 숫자 입력
    - 종료 여부 입력 
#### outputView
    - 스트라이크, 볼 개수 출력
    - 종료 알림 출력
