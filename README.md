# 숫자 야구 게임
## 진행 방법
* 숫자 야구 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 과제 제출 과정
* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/precourse)

## 기능 목록
### 1. View
* 숫자를 입력해주세요
* 게임을 새로 시작하려면 1, 종료하려면 2를 입력하세요.

### 2. Controller
* 게임을 시작하는 API
* 게임을 종료하는 API
* 숫자를 입력받고 결과를 알려주는 API
* 입력받은 값이 올바른 값인 지 체크

### 3. Model
#### 3-1. Game
* 서로다른 세 개의 숫자로 타겟값을 만드는 기능
* 입력값에 대한 결과를 만드는 기능
#### 3-2. Input
* 자신의 타겟값에 대한 결과가 스트라이크인지 볼인지 맞추는 기능
