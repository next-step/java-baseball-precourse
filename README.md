# 숫자 야구 게임
## 진행 방법
* 숫자 야구 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 과제 제출 과정
* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/precourse)


## 기능 목록
- [X] 컴퓨터 플레이어 
- [X] 정답
  - [X] 1~9로 이루어진 3자리 수 생성
  - [X] 숫자에 대한 결과 판독
    - [X] 사용자가 잘못된 값 입력시 에러처리 후 계속 진행할 수 있도록 처리 ([ERROR]로 시작하는 에러 메시지)
    - [X] 같은 자리 스트라이크
    - [X] 다른 자리 있을시 볼
    - [X] 전혀 없을 경우 포볼 or 낫싱
- [X] 체크 값
- [X] 플레이어로부터 입력받기
- [X] 게임 종료시, 다시 시작 or 종료 선택
