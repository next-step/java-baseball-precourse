# 숫자 야구 게임

## 진행 방법

* 숫자 야구 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 과제 제출 과정

* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/precourse)

## 기능목록

1. (컴퓨터)중복되지 않는 임의의 수 3개를 선택한다.
2. 게임 플레이어는 컴퓨터가 생각하고 있는 3개의 숫자를 입력한다.
3. 같은 수가 같은 자리에 있는지 체크하여 스트라이크 카운트를 측정하고 힌트에 포함한다.
4. 같은 수이며 자리가 다른 경우 볼 카운트를 측정하고 힌트에 포함한다.
5. 같은 수가 전혀 없으면 포볼 또는 낫싱이란 힌트를 준다.
6. (3, 4, 5)번의 기능이 3스트라이크가 나올때까지 반복 되도록 만든다.
7. 3스트라이크가 나올 경우 게임을 새로 시작 할 건지 종료 할 건지 입력받아 선택한다.
