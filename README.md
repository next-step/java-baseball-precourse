# 숫자 야구 게임
## 진행 방법
* 숫자 야구 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 과제 제출 과정
* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/precourse)

## 기능구현
* 1 ~ 9까지 서로 다른 임의의 수 생성 기능 개발
* 사용자로부터 값을 입력받는 기능 개발
* 사용자가 입력한 값과 컴퓨터가 생성한 임의의 수 비교 기능 개발
* 비교 결과 출력 기능 개발
* 게임을 종료하거나 다시시작하는 기능 개발
