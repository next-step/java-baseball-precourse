# 숫자 야구 게임
## 진행 방법
* 숫자 야구 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 과제 제출 과정
* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/precourse)

## 기능목록
1. 정답 번호 생성
   - 3개의 숫자
   - 1 ~ 9까지
2. 사용자 입력
   - 유효성 검사
       + 3개의 숫자
       + 숫자만 입력이 되었는지
       + 중복된 숫자가 있는지
   - 유효성 검사를 통과하지 못하면 에러 메시지 출력 후 다시 입력받게 함

3. 정답과 사용자 입력값 비교하여 결과 생성
   - 번호의 위치가 일치하면 스트라이크 카운트 증가
   - 번호가 있지만 위치가 일치하지 않으면 볼 증가
   - 스트라이크, 볼이 존재하지 않으면 낫싱

4. (추가) 비교 결과를 사용자에게 출력

5. 스트라이크가 될 때 까지 2 ~ 4번 반복

6. 게임을 다시 시작할지, 종료할지를 값을 입력받아 처리
