# 숫자 야구 게임

---------------------------------------
### 숫자 야구 게임 룰
> 기본적으로 1부터 9까지 서로 다른 수로 이루어진 3자리의 수를 맞추는 게임으로써 같은 수가 같은 자리에 있으면 스트라이크, 다른 자리에 있으면 볼, 같은 수가 없으면 포볼 또는 낫싱 이란 힌트를 얻고 이용하여 먼저 상대방의 수를 맞추면 승리하는 게임이다.

### 숫자 야구 게임 기능 목록

- [✔️️] 상대방의 임의의 수 `3자리`를 만드는 기능
- [✔️️] 상대방의 임의의 수 `3자리`와 비교할 수를 입력 받는 기능
    - 예외목록
        - `3자리` 수 또는 숫자 입력이 아닌 경우 `IllegalArgumentException` 발생과 프로그램이 종료된다.
- [✔️️] 상대방의 임의 수 자리와 입력 받은 수를 비교하는 기능
- [✔️️] 비교 한 값의 결과 값을 출력
- [✔️️] 게임이 완료 된 경우 지속적으로 게임 진행 할지 여부 선택
    - `진행 수락 시` 1번 기능부터 다시 시작
    - `진행 거부 시` 게임 완전히 종료
