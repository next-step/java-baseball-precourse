# 숫자 야구 게임
## 진행 방법
* 숫자 야구 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 과제 제출 과정
* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/precourse)

## 기능 구현 목록

1. 게임 시작
   - <del>서로다른 3자리의 숫자가 랜덤으로 생성된다.</del>
2. 플레이어 숫자 입력
   - 서로다른 3자리의 숫자를 입력받는다.
   - 입력받은 숫자를 검증한다(1~9 내의 서로다른 3자리의 숫자)
     - 잘못된 숫자를 입력받을 경우 플레이어에게 에러 메세지를 출력한다. 
3. 플레이어의 숫자와 랜덤으로 생성된 숫자를 비교한다.
   - 같은 숫자가 같은 자리에 있으면 스트라이크
   - 같은 숫자라 다른 자리에 있으면 볼
   - 스트라이크와 볼이 전부 없다면 낫싱
   - 결과를 플레이어에게 전달한다.
   - 모든 숫자가 스트라이크일 시 게임 종료 메세지를 출력한다.
4. 게임종료후 플레이어에게 계속 진행 여부를 입력받는다.
   - 입력받은 숫자를 검증한다(1, 2) 이외의 입력을 받을 경후 에러 메세지를 출력한다.
   - 게임 시작(1)을 입력받으면 1로 돌아간다.
   - 게임 종료(2)를 입력받으면 프로그램을 종료한다.
