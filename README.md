# 숫자 야구 게임
## 프로세스
1. 랜덤 난수 생성
2. 숫자 입력
    - 입력값 정합성 검증
        - 자리수(3자리)
        - 입력갑(1~9, 서로 다른 수)
3. 숫자 비교
    - 스트라이크 판별(같은 수, 같은 자리)
    - 볼 판별(같은 수, 다른 자리)
    - 낫싱/포볼 판별(수가 없음)
4. 결과를 리턴
5. 3개의 숫자를 모두 맞춰 3스트라이크가 나올때 까지 반복 진행

## 구현 기능 목록
1. 랜덤 난수 생성 - nextstep 제공하는 API사용
2. 숫자 입력 - nextstap 제공하는 API사용
3. 입력 숫자 정합성 검증 기능
   - 자리수가 3을 초과하는 경우 처리를 어떻게 할 것인가? 요구사항이 명확하지 않으므로,
     
      **- 3자리가 입력되지 않으면 재입력을 받는다. (이것으로 진행)**
     
      ~~- 첫번째 자리부터 뒤로 3자리만 처리한다.~~
     
      ~~- 마지막 자리부터 앞으로 3자리만 처리한다.~~
   
3. 숫자 비교 기능
    - 스트라이크 판별 기능
    - 볼 판별 기능
    - 낫싱 판별 기능
3. 숫자 비교 결과 반환 기능
