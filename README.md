# 숫자 야구 게임
## 진행 방법
* 숫자 야구 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 과제 제출 과정
* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/precourse)

## 요구사항
- [X] 3자리 수를 생성한다
  - [X] 같은 수가 같은 자리에 있으면 스트라이크로 판정한다
  - [X] 같은 수가 다른 자리에 있으면 볼로 판정한다
  - [X] 같은 수가 전혀 없으면 낫싱으로 판정한다
  - [X] 외부로부터 값을 받아서 생성한다
  - [X] 3개의 숫자가 중복되지 않아야 한다
  - [X] 한자리의 수를 나타내는 공을 생성한다
    - [X] 수가 1 ~ 9 사이여야 한다
    - [X] 같은 수로 생성한 공은 항상 같은 공으로 취급된다
- [ ] 게임을 진행한다
  - [ ] 컴퓨터가 선택한 3개의 숫자를 스트라이크로 모두 맞춰 3스트라이크가 되면 게임이 종료된다
  - [ ] 3스트라이크로 게임이 종료된 후 게임을 다시 시작하거나 종료한다
    - [ ] 1을 입력받으면 게임을 새로 시작한다
    - [ ] 2를 입력받으면 게임을 완전히 종료한다
- [ ] 예외사항에 대해 처리한다
  - [ ] 사용자가 잘못된 값을 입력하면 에러 메시지를 출력하고 게임을 계속 진행한다
