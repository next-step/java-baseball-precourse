# 숫자 야구 게임
## 진행 방법
* 숫자 야구 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 과제 제출 과정
* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/precourse)

## 구현할 기능 목록
* 1~9 사이의 서로 다른 임의의 숫자를 세개 생성한다.
* 사용자의 입력값을 받는다.
    * 입력값 검증
        * 입력값은 숫자 형식이어야 한다.
        * 입력값은 1부터 9 사이어야 한다.
    * 입력값을 숫자로 형변환한다.
    * 검증 결과를 출력한다.
* 사용자에게 중복되지 않은 3개의 숫자를 입력 받는다.
    * 입력값 검증
        * 입력값은 3개이어야 한다.
        * 입력값은 중복되면 안 된다.
    * 검증 결과를 출력한다.
* 랜덤값과 사용자 입력값을 비교하여 결과를 낸다.
    * 스트라이크 개수와 볼 개수를 계산한다.
    * 결과문(스트라이크, 볼, 낫싱)을 얻는다.
    * 성공 여부를 판단한다.
    * 결과를 출력한다.
* 재입력 여부를 판단한다.
    * 실패 시 재입력 받는다.
    * 성공 시 다시 시작 여부를 받는다.
        * 입력값은 1, 2만 가능하다.
        * 입력값이 1이면 게임을 다시 시작한다.
        * 입력값이 2이면 게임을 종료한다.
