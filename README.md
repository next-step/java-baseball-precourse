# 숫자 야구 게임
## 진행 방법
* 숫자 야구 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

### 기능목록
    - 숫자입력받기
      - 입력받은 숫자를 String > int 로 바꾸기 
    - 입력값 유효성 체크
    - 입력값을 체크한 결과값 출력하기 
    - 입력받은 숫자로 응답값 산출하기
        - 스트라이크 개수 판별하기(같은위치, 같은 값)
        - 볼 개수 판별하기 (다른 위치, 같은 값)


응답값 케이스 

- 완전일치 (정답/게임종료)
- 완전 불일치 (낫싱)
- 부분일치 
  - 스트라이크(같은위치 같은값)
  - 볼(다른위치 같은값)
