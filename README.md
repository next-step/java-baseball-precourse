# 숫자 야구 게임

## 기능 구현 목록
> 1부터 9까지 서로 다른 숫자로 이루어진 3자리의 수를 찾추는 게임이다.
1. 3자리의 겹치지 않는 랜덤 숫자를 정한다.
2. 플레이어에게 맞추고자 하는 예상 숫자 3자리를 입력받는다.
   - 아래의 경우 해당 메세지를 출력 후 다시 `2.` 로 돌아간다.
     - (3자리 숫자를 입력하지 않았다면) `"[ERROR] 3자리 숫자가 아닙니다."` 라는 에러 메세지 출력
     - (입력한 숫자에 0이 포함되어 있다면) `"[ERROR] 0이 포함되면 안됩니다."` 라는 에러 메세지 출력
3. 정해진 랜덤 숫자와 플레이어가 제시한 숫자를 비교한 결과를 출력한다.
   - 아래의 경우 해당 메세지를 출력 후 다시 `2.` 로 돌아간다.
     - (비교하는 두 숫자의 위치와 숫자가 같은 숫자가 있다면) `"{해당 숫자의 개수}스트라이크"` 를 출력 
     - (비교하는 두 숫자의 위치는 다르지만 같은 숫자가 있다면) `"{해당 숫자의 개수}볼"` 를 출력 
     - (비교하는 두 숫자의 위치와 숫자가 같은 숫자와 위치는 다르지만 같은 숫자 둘다 있다면) `"{위치와 숫자가 같은 숫자의 개수}스트라이크 {위치는 다르지만 같은 숫자의 개수}볼"` 를 출력
     - (위 경우에 해당하는 숫자가 없다면) `"낫싱"` 를 출력 
   - (비교하는 두 숫자가 모두 같다면) `"3개의 숫자를 모두 맞히셨습니다! 게임 끝"` 을 출력한다.
5. 사용자에게 게임을 새로 시작하려면 1, 종료하려면 2를 입력받는다.
   - (1 또는 2 이외의 값을 입력한 경우) `"[ERROR] 잘못된 값을 입력하였습니다."` 출력 후 다시 `4.` 로 돌아간다.
   - (1을 입력했다면) 다시 `1.` 로 돌아간다.
   - (2를 입력했다면) 프로그램을 종료한다.
