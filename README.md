# 숫자 야구 게임
## 기능 목록
- [x] 1부터 9까지의 서로 다른 수로 구성된 세자리 수를 생성한다 
- [x] 사용자로부터 세자리 수를 입력받는다
- [x] 사용자 입력과 일치하는 자릿수를 계산한다
- [x] 일치하는 자릿수를 바탕으로 힌트를 출력한다
  - [x] 같은 수가 같은 자리에 있으면 "스트라이크"
  - [x] 같은 수가 다른 자리에 있으면 "볼"
  - [x] 같은 수가 전혀 없으면 "낫싱"
  - [x] 사용자가 잘못된 값을 입력하면 "ERROR"
- [x] 사용자가 세자리 수를 모두 맞히면 게임을 종료한다

## 프로그램 예시
```
숫자를 입력해주세요 : 123
1 스트라이크 1볼
숫자를 입력해주세요 : 145
1볼
숫자를 입력해주세요 : 671
2볼
숫자를 입력해주세요 : 216
1스트라이크
숫자를 입력해주세요 : 713
3스트라이크
3개의 숫자를 모두 맞히셨습니다! 게임 끝
게임을 새로 시작하려면 1, 종료하려면 2를 입력하세요.
1
숫자를 입력해주세요 : 123
1볼
```
