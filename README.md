# 숫자 야구 게임
## 진행 방법
* 숫자 야구 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 과제 제출 과정
* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/precourse)

## 기능 요구사항 정리
### 플레이어 
* 플레이어는 1부터 9까지 서로 다른 수로 이루어진 3자리 수를 입력한다.

### 상대방 (컴퓨터)
* 상대방은 1부터 9까지 서로 다른 수로 이루어진 3자리 수를 정답으로 정한다.
* 정답은 랜덤하게 결정된다.

### 확인자
* 확인자는 플레이어가 입력한 3자리와 컴퓨터가 설정한 정답 3자리를 비교한다
* 비교 결과 같은 수가 같은자리에 있으면 스트라이크, 다른 자리에 있으면 볼, 같은 수가 전혀없으면 포볼 또는 낫싱이란 힌트를 제공한다.

### 게임
* 게임은 플레이어가 상대방의 숫자를 맞추면 종료한다.
* 게임이 종료 될 때까지 플레이어에게 입력값을 받는다
* 게임을 종료한 후에 게임을 다시 시작하거나 완전히 종료할 수 있다.
