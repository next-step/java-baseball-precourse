# 숫자 야구 게임
## 진행 방법
* 숫자 야구 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 과제 제출 과정
* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/precourse)


## 구현 기능 목록

---

- 게임 시작 
- 게임 종료
- 정답 생성
- 유효성 검사
- 입력 시도 
- 정답 확인
- 힌트 제공
- 승리
- 입력 시도 횟수 저장

## 구현 객체

--- 
1. Game
    + 한번의 게임을 의미 
    + 입력받은 값과 정답을 비교하여 정답 여부 판단
    + 한번의 입력 시도 횟수 저장

1. TryResult
   + 입력 시도의 결과를 저장

1. Computer
   + 게임의 시작과, 종료를 담당 
   + 매 게임시 정답을 생성하여 Game에 전달 
   + 게임 승리시 재시작 여부를 확인
   
1. Validator
   + 입력 값의 유효성을 검사
   + 공백 및 널 검사 
   + 정규식 검사
   + 숫자 파싱검사
   + 중복 검사
   + 유효성이 맞지 않을시 에러 생성
