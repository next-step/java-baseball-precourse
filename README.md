# 숫자 야구 게임
## 컴퓨터
* 랜덤하면서 서로 겹치지 않는 세 숫자를 1-9 범위 내에서 고를 수 있다.
* 라운드가 시작될 때마다 새로운 숫자를 고른다.
* 인덱스 번호를 통해 해당 숫자가 컴퓨터가 가진 숫자에 포함되어 있는지 확인할 수 있다.
* 하나의 숫자가 컴퓨터가 가진 숫자에 포함되어 있는지 확인할 수 있다.

## 사용자
* 1-9 범위 내의 세 숫자를 문자열로 입력할 수 있다.
* 잘못된 숫자를 입력하면 IllegalArgumentException이 발생한다.
* 인덱스 번호를 통해 해당 숫자가 사용자가 가진 숫자에 포함되어 있는지 확인할 수 있다.

## 점수
* 라운드가 시작될 때마다 점수는 초기화된다.
* 스트라이크와 볼, 낫싱 횟수를 판단한다.
* 특정 인덱스의 숫자가 스트라이크이면 스트라이크 횟수를 증가한다.
* 특정 인덱스의 숫자가 스트라이크가 아니면 볼인지 확인한다.
* 해당 라운드의 점수를 출력할 수 있다.

## 게임
* 게임을 시작하면 초기화 작업을 거친 후 라운드를 시작한다.
* 각 라운드는 모든 숫자가 스트라이크일 때까지 반복한다. 
* 모든 숫자가 스트라이크이면 종료 작업을 거친다. 이때 사용자가 1을 입력하면 게임을 종료하고, 2를 입력하면 새로운 라운드를 시작한다.
