# 숫자 야구 게임
## 진행 방법
* 숫자 야구 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 과제 제출 과정
* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/precourse)

# 룰
	- 1부터 9까지 서로 다른 수로 이루어진 3자리의 수를 맞추는 게임

# 구현할 기능 목록

## 컨트롤러 ( 진행자 역할 ) 
	- 전체 플로우 진행
	- 사용자로부터 입력값을 받는 기능
	- 결과 로직  
	- 결과 출력
	
## 컴퓨터 ( 상대방 ) 
	- 랜덤 수 생성
	- Strike / Ball 카운트 로직
	
## 체크 ( 사용자의 입력값 확인 ) 
	- 사용자에 대한 입력값 체크
		* empty 확인
		* 숫자인지 확인
		* 자리수 확인
		* 같은값 있는지 확인

## 기타
	- String / Integer 체크 (valid) 
	- 전역변수 저장 클레스
	- 테스트 프로그램 작성
