# 숫자 야구 게임

## 기능 요구사항 📑
* 컴퓨터는 서로 다른 숫자 3개를 선택 할 수 있다.
  - [x] 랜덤함수를 이용하여 숫자 3개를 선택한다.
  - [x] 숫자의 범위가 1~9 사이이다.
  - [x] 숫자에 중복이 있는 경우가 생기면 안된다.
  

* 플레이어는 숫자를 입력 할 수 있다.
  - [x] console을 사용하여 입력 받는다.
  - [x] 문자열 형태의 숫자를 숫자 리스트로 변환한다.
  
  
* 게임 결과를 규칙에 따라 판정
  - [x] 특정 자리에 특정 숫자가 있는 경우 카운트
  - [x] 특정 숫자가 포함된 경우 카운트
  * 스트라이크
    - [x] 특정 자리에 특정 숫자가 있는 경우 카운트가 스트라이크 수
  * 볼
    - [x] 숫자만 같은 카운트 - 스크라이크 수
  * 낫싱
    - [x] 숫자만 같은 경우 카운트가 0 인 경우
  
  
* 야구 숫자에 대한 검증
  - [x] 입력 된 숫자가 없으면 NoSuchElementException가 발생 한다.
  - [x] 숫자의 범위가 1~9 사이이다.
  - [x] 숫자에 중복이 있는 경우가 생기면 안된다.
  - [x] 입력된 숫자는 3자리 여야 한다.
  - [x] 검증 예외 상황 발생시 throw 한다.
  
  
* 게임실행 컨트롤
  * 게임을 다시 시작하거나 완전히 종료할 수 있다.
    - [ ] 1이 입력되면 새로 시작한다.
    - [ ] 2가 입력되면 게임이 종료된다.
  * 게임 종료 조건을 판단할 수 있다. 
    - [ ] 사용자가 입력한 숫자들과 컴퓨터가 선택한 숫자가 완벽히 일치하면(3스트라이크) 게임이 종료된다.
  * 사용자가 잘못된 값을 입력할 경우 게임을 계속 진행할 수 있어야 한다.
    - [ ] [ERROR]로 시작하는 에러 메시지를 출력한다.
    - [ ] 입력은 계속 받을 수 있어야 한다.
