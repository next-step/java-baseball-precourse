# java-baseball-precourse

## 구현할 기능
- 랜덤 숫자 생성(CreateRandom)
  - 중복 제거
  - 글자 수 3개
- 사용자 입력 받기(Player)
  - 글자 수 3개인지 확인
  - 정수인지 확인
  - 중복 확인
- 스트라이크, 볼 판정(Judgement.judge)
  - 숫자 + 자리수 맡으면 스트라이크
  - 숫자만 맞으면 볼
  - 다 틀리면 낫싱
- 성공 판정(Judgement.result)
  - 스트라이크 3개면 성공
  - 사용자에게 게임 지속 여부 확인
    - 입력 값이 1이면 계속(랜덤 값 재생성)
    - 입력 값이 2이면 종료
