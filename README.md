# 숫자 야구 게임
## 진행 방법
* 숫자 야구 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 과제 제출 과정
* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/precourse)


## 기능 목록

### 서로 다른 3개의 숫자로 이루어진 정답 생성
- 이미 정해진 숫자들과는 다른 새로운 랜덤 숫자 생성

### 사용자 추측 번호
- 3자리 숫자 입력 받기
- 판정
    - Strike
    - Ball
    - 낫싱 

### 게임 UI 개발
- 시작
- 재시작 권유
- 종료
