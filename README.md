# 숫자 야구 게임
## 진행 방법
* 숫자 야구 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 과제 제출 과정
* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/precourse)



## 기능

### Score

| 기능                      | 설명         |
| ------------------------- | ------------ |
| start                     | 게임 시작    |
| finish                    | 게임 종료    |
| restart                   | 게임 재시작  |
| createBallCount           | 공 생성      |
| input                     | 값 입력      |
| valivalidationSelectBalls | 입력 값 검증 |
| compareValue              | 값 비교      |



### Message

| 기능   | 설명             |
| ------ | ---------------- |
| start  | 게임 시작        |
| finish | 게임 종료        |
| error  | 사용자 값 오입력 |
| result | 게임 결과        |
