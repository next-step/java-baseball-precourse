## 구현할 기능 목록
1. 컴퓨터의 랜덤한 세자리 수 만들기
 - 서로 다른 수 인지 확인
 - 게임 시작
2. 사용자 입력받기
 - 입력 값이 잘못된경우
   1) 숫자가 아닌경우
   2) 양수가 아닌경우
 - IllegalArgumentException 발생 후 강제 종료
 - 입력 값이 이미 한번 체크 한 수라면 중복 표시
 - 입력 값이 서로 다른 수로 이뤄진 3자리 수가 아니라면 유효한 값이 아니라는 표시
4. 정답에 따라 스트라이크와 볼 갯수 처리하기
 - 볼, 스트라이크, 낫싱
5. 처리 결과에 따라 다음 행동 처리
 - 처리된 스트라이크, 볼 갯수 표시
6. 정답을 다 맞춰 종료 후 게임 재시작, 종료 선택 처리
7. 종료시 어플리케이션 종료
