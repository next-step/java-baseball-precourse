
# 숫자 야구 게임
## 기능 요구 사항

* 1-9의 3자리의 난수 생성
  * Randoms api를 이용
* 사용자로부터 값을 입력받음
  * Console api를 이용
  * 만약 3자리 이상이라면 IllegalArgumentException 발생
* 입력받은 값과 3자리 난수 와 입력받은 값을 비교
  * 자리와 값이 같다면 스트라이크
  * 값은 존재하나 자리가 틀리다면 볼
* 모두 맞았다면 3스트라이크, 볼이 존재한다면 볼, 스트라이크, 틀렸다면 낫싱을 출력
* 3스트라이크시 1 혹은 2를 입력받아 게임을 계속할지 그만둘지 정함
