## 기능 목록
- [x] 💻 : 1에서 9까지 서로 다른 임의의 수 3개 선택
- [x] 🤹 : 컴퓨터가 생각하고 있는 3개의 숫자 입력
  - 잘못된 값을 입력할 경우 IllegalArgumentException 발생 후 종료
- [ ] 💻 : 입력한 숫자에 대한 결과 출력
  - 스트라이크 : 같은 수, 같은 자리
  - 볼 : 같은 수, 다른 자리
  - 낫싱 : 같은 수가 없음
- [ ] 🤹 : 선택한 3개의 숫자를 모두 맞히면 게임 종료
  - 1 : 게임을 다시 시작
  - 2 : 게임을 완전히 종료

## 프로그램 실행 예제
```
숫자를 입력해주세요 : 123
1볼 1스트라이크
숫자를 입력해주세요 : 145
1볼
숫자를 입력해주세요 : 671
2볼
숫자를 입력해주세요 : 216
1스트라이크
숫자를 입력해주세요 : 713
3스트라이크
3개의 숫자를 모두 맞히셨습니다! 게임 종료
게임을 새로 시작하려면 1, 종료하려면 2를 입력하세요.
1
숫자를 입력해주세요 : 123
1볼
...
```
