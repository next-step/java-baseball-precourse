## 기능 목록
- [ ] 🤹 : 게임 시작
- [x] 💻 : 1에서 9까지 서로 다른 임의의 수 3개를 선택
- [ ] 🤹 : 컴퓨터가 생각하고 있는 3개의 숫자 입력
    - [ ] 잘못된 값을 입력할 경우 IllegalArgumentException 발생 후 종료
- [ ] 💻 : 게임 플레이어가 입력한 숫자에 대한 결과 출력
    - [ ] 스트라이크 : 같은 숫자 & 같은 자리
    - [ ] 볼 : 같은 숫자 & 다른 자리
    - [ ] 낫싱 : 같은 숫자가 없음
- [ ] 💻 : 선택한 3개의 숫자를 모두 맞히면 게임 종료
- [ ] 🤹 : 게임 시작 여부 선택
    - 1 : 게임을 다시 시작
    - 2 : 게임을 완전히 종료
