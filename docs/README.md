## 구현할 기능
* UI  
  * 입력
    * 숫자야구 게임을 위한 3자리의 수를 입력한다.
    * 잘못된 값을 입력할 경우를 예외처리한다.
  * 출력
    * 입력한 숫자에 대한 결과를 출력한다.
* 게임 규칙
  * 상대방(컴퓨터)의 수 생성
    * 1에서 9까지 서로 다른 임의 수를 생성한다.
  * 낫싱
    * 같은 수가 전혀 없으면 낫싱이라는 힌트를 제공한다.
  * 스트라이크
    * 같은 수가 같은 자리에 있는 숫자를 카운트한다.
  * 볼
    * 같은 수지만 다른 자리에있는 숫자를 카운트한다.
  * 게임을 종료한 후 게임을 다시 시작하거나 종료한다.
