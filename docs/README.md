# 기능 목록

- 사용자는 1 ~ 9 사이 숫자 3개를 입력할 수 있어야 한다.
    - 잘못된 값을 입력할 경우 예외(IllegalArgumentException)를 발생시킨 후 애플리케이션이 종료되어야 한다.
        - 볼을 3개보다 적게 또는 많게 입력했을 경우
        - 볼 숫자가 1 ~ 9 사이가 아닌 경우
        - 게임 재시작 플래그 1,2가 아닌 수를 입력했을 경우
- 컴퓨터는 1 ~ 9 사이 숫자를 서로 다른 임의의 수(난수)를 3개 생성해야 한다.
- 사용자가 입력한 3개의 숫자와 컴퓨터가 생성한 3개의 숫자를 비교하여 결과를 계산해야 한다.
    - 같은 자리 같은 숫자 : 스트라이크
    - 다른 자리 같은 숫자 : 볼
    - 아예 없는 경우 : 낫싱
- 컴퓨터의 결과를 출력할 수 있어야 한다.
- 결과가 3스트라이크가 나올 때까지 위 과정을 반복할 수 있어야 한다.
- 3스트라이크가 나올 경우 게임이 종료된 후 다시 시작 및 완전 종료가 가능해야 한다.
