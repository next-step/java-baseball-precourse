# Bulls and cows

## 기능 목록

1. 게임 진행을 선택한다.
   1. 게임을 진행한다.
   2. 유저가 종료한다.
   3. IllegalArgumentException 던진다.
2. 게임을 진행한다.
   1. 잘못된 입력인 경우 종료한다.
      1. 숫자이며 1 ~ 9 범위이며 모두 다른 값이다. (Ex, 123, 456, 781)
   2. 입력에 대한 결과를 보여준다.
      1. 정답인 경우 1번으로 돌아간다.
      2. 다음 시도에 대한 입력을 받는다.

## 객체의 역할과 책임

- BullsAndCowsGameController
  - 사용자 입력을 바탕으로 다른 모델에 명령을 보낸다.
  - 명령에 대한 반환값을 바탕으로 결과를 출력한다.
- BullsAndCowsGame
  - 실제 게임에 대한 상태를 관리한다.
  - 정답을 갖고 있으며 입력에 대한 결과를 반환한다.
- Trial
  - 사용자의 입력
- Answer
  - 정답
- Result
  - 입력에 대한 결과
- BullsAndCowsView
  - 입력에 대한 결과를 사용자에게 보여준다.
