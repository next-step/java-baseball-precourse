# 숫자 야구 게임
## 기능 목록

* 상대방 역활
  * 임의의 수 서로 다른 3자리 숫자를 생성 한다.
  * 숫자는 0이 아닌 양수 여야 한다.
* 플레이어 역활
  * 서로 다른 3자리 숫자를 입력 하여 상대방의 숫자를 맞춘다.
* 입력
  * 입력값을 받아온다.
* 결과 출력
  * 상대방 숫자를 전부 맞추면 종료 
  * 상대방의 숫자가 한개라도 다르면 힌트를 제공한다.
  * 힌트가 제공 된후 다시 입력 받는다.
* 힌트[예] 상대방(컴퓨터)의수가425일때
  * 123을제시한경우:1스트라이크
  * 456을제시한경우:1볼1스트라이크
  * 789를제시한경우:낫싱
