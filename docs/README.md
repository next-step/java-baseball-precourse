
# 기능

## 개요
* 사용자로부터 3자리의 수를 입력받아 무작위로 생성된 숫자 3개를 맞히면 게임은 종료된다.

## 상세
[시작 - 무작위 숫자 생성 - 사용자 입력 - 판정 - 게임 재시작 또는 종료] 의 순서를 가진다.

* 1부터 9까지 서로 다른 수로 이루어진 3자리의 수를 무작위로 생성한다.
* 사용자로부터 3자리의 숫자를 입력받는다.
* 같은 숫자가 같은 자리에 있으면 스트라이크, 다른 자리에 있으면 볼, 같은 숫자가 하나도 없으면 낫싱을 출력한다.
    * e.g.
        * 2볼 1스트라이크
        * 3스트라이크
        * 낫싱
* 게임이 종료되면 사용자의 선택으로 새 게임을 시작하거나 앱을 종료할 수 있다.
* 잘못된 입력의 경우 예외를 발생하고 종료된다.
