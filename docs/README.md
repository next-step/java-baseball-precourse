## 기능 요구사항

1. 플레이어는 게임을 시작한다.
   - [x]  1부터 9까지 서로 다른 수로 이루어진 임의의 3자리 숫자가 생성된다.
2. 플레이어는 임의의 수를 입력할 수 있다.
   - [x]  플레이어가 잘못된 값을 입력하는 경우, IllegalArgumentException 이 발생하고, 게임은 종료된다.
      - [x]  3자리까지 입력 가능하다.
      - [x]  각 자리에는 1과 9 사이의 숫자만 입력 가능하다.
3. 플레이어는 입력한 숫자에 대한 결과를 볼 수 있다.
   - [x]  게임에서 승리하면, 게임은 종료된다.
      - [x]  3자리 숫자 모두 일치하는 경우 (3 스트라이크) 승리한다.
   - [ ]  게임에서 승리하지 못하면, 플레이어는 결과를 보고 임의의 수를 다시 입력한다.
      - [ ]  같은 수가 같은 자리에 있으면 스트라이크
      - [ ]  같은 수가 다른 자리에 있으면 볼
      - [ ]  같은 수가 전혀 없으면 낫싱
4. 플레이어는 게임 종료 후, 새 게임 시작 여부를 결정할 수 있다.
   - [ ]  1은 시작, 2는 종료
