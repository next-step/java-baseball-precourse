# 우아한테크캠프 PRO 숫자 야구게임 미션

## 기능 명세
- 컴퓨터가 제시한 3자리 숫자가 존재하며, 클라이언트가 3자리의 숫자를 입력
  - 동일한 숫자가 컴퓨터 기록한 위치와 클라이언트가 입력한 숫자의 자릿수와 같으면 1 strike
  - 동일한 숫자가 3자리내에 각기 포함되더라도, 서로 자리수가 다르면 1 ball
  - 클라이언트가 입력한 각 숫자가 컴퓨터가 제시한 3자리의 숫자 중 자릿수 무관하게 매핑되지 않는다면 nothing 처리
- 사용자의 오입력은 IllegalArgumentException을 발생시킨 후 종료되어야 한다.

## 구현 기능 목록
- [x] 컴퓨터 1~9 사이의 서로다른 3자리 수 생성
  - [x] missionutils 이용하여 Randoms 3자리 숫자값 생성
- [x] 유저의 3자리 수 입력받는 기능 개발
  - [x] 1~9 사이의 3자리 숫자 입력 여부 검증 필요
  - [x] 공백, 특수기호, 문자 등 사용 불가
  - [x] 중복되는 숫자 사용 불가
- [x] 스트라이크 판별 기능 개발
  - [x] 컴퓨터와 클라이언트가 입력한 각 세자리 숫자중 동일 인덱스에 동일 숫자 위치하는 경우 strike count (+1)
- [x] 볼 판별 기능 개발
  - [x] 컴퓨터와 클라이언트가 입력한 각 세자리 숫자중 인덱스는 다르나 동일 숫자 포함하는 경우 ball count (+1)
- [x] Nothing 판별 기능 개발
  - [x] 클라이언트가 입력한 세자리 숫자중 각 숫자가 모두 컴퓨터가 입력한 세자리 숫자에 포함되지 않는 경우
- [] View 에서 스트라이크/볼/낫싱 결과를 전달받아 출력
- [] 게임 재시작 및 종료 여부 입력받아 게임 진행 여부 결정하는 로직

## 주의 사항
- 요구 명세에 벗어나는 입력은 예외처리 되어야 함.
- UI의 변경 사항이 발생하더라도 도메인에 영향을 끼치지 않아야함.
- JAVA8 환경에서 컴파일, 빌드 등이 가능해야 함.
- 랜덤 값 생성 및 입력 받는 것은 nextstep 의 missionutils를 이용한다.
