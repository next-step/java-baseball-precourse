# Bulls and cows

## 기능 목록

1. 게임 진행을 선택한다.
   1. 게임을 진행한다.
   2. 유저가 종료한다.
   3. IllegalArgumentException 던진다.
2. 게임을 진행한다.
   1. 잘못된 입력인 경우 종료한다.
      1. 숫자이며 1 ~ 9 범위이며 모두 다른 값이다. (Ex, 123, 456, 781)
   2. 입력에 대한 결과를 보여준다.
      1. 정답인 경우 1번으로 돌아간다.
      2. 다음 시도에 대한 입력을 받는다.

## 객체의 역할과 책임

- BullsAndCowsGameController
  - 사용자 입력을 바탕으로 비즈니스 모델에 명령을 보낸다.
  - 반환된 결과값을 바탕으로 결과를 출력한다.
  - 올바른 BullsAndCowsGame 객체를 생성한다.
- BullsAndCowsGame
  - 실제 게임에 대한 상태를 관리한다.
  - 정답을 갖고 있으며 입력에 대한 결과를 반환한다.
  - 도전하면 Answer 객체에 도전을 위임한다.
- Trial
  - 정답을 풀기 위한 사용자의 시도
  - 3 수 모두 다름을 보장한다. (입력 형식이 맞지 않으면 예외 발생)
- Answer
  - 사용자가 맞출 3 숫자
  - 3 수 모두 다름을 보장한다. (입력 형식이 맞지 않으면 예외 발생)
- Result
  - 시도에 대한 결과(힌트)
- BullsAndCowsView
  - 입력에 대한 결과를 사용자에게 보여준다.

## 고민사항

- 숫자의 범위와 숫자의 갯수 값이 변경될 수 있는 요구 사항이 나중에 생기면 Game 객체를 생성하는 팩토리 클래스를 만들면 좋을것 같습니다.
- 서로 다른 3 숫자를 생성하기 위해 재귀 호출을 사용하고 있어 아쉬운 점이 있습니다.
