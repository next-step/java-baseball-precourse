# 1. 요구 사항 분석 및 기능 도출

## 1.1. 3자리 수 랜덤 생성 기능
- 게임 시작시 자동으로 생성하며, 랜덤 수는 1-9 사이의 수를 생성한다.
- missionutils.Randoms의 pickNumberInRange() 활용

## 1.2. 3자리 수 입력 기능

- 사용자가 야구 게임의 수를 맞추기 위해, 값을 입력하는 부분이다.
- missionutils.Console의 readLine() 활용

## 1.3. 메시지 출력 기능

- 사용자가 입력한 값을 가지고 메시지를 출력한다. 
- System.io, System.out을 통한 메시지 구현

## 1.4. 게임 시작 및 종료 기능

- 프로그램을 시작하면 게임이 자동으로 시작되고,
  숫자를 맞추고 게임이 종료되면 게임을 시작할 수 있는 기능이 필요하다.
