## 기능 구현
1. 컴퓨터는 서로 다른 숫자 3개를 선택한다.
2. 플레이어는 컴퓨터가 생각하고 있는 3자리 숫자를 입력한다.
3. 입력한 숫자에 대한 결과를 비교한다.
4. 비교 결과에 따른 출력 결과를 리턴한다.
5. 출력 결과를 출력한다.
6. 비교 결과가 3개가 모두 맞다면 게임을 종료한다.
7. 틀린 것이 있다면 플레이어 입력으로 돌아간다.
8. 게임을 다시 시작할수 있다.
9. 게임을 완전히 종료한다.
10. 사용자가 잘못된 값을 입력했는지 점검한다.
11. 잘못된 값을 입력했다면 IllegalArgumentException 오류를 발생시킨다.
12. 어플리케이션을 종료한다.
