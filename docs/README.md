# 숫자 야구 게임

### 기능 목록
- [X] 게임 플레이어에게 숫자를 입력 받는다.
  - [X] 숫자는 1부터 9까지 서로 다른 수로 이루어진 3자리의 수만 입력 가능하다.
  - [X] 게임을 종료한 후 게임을 다시 시작하려면 1, 완전히 종료하려면 2를 입력한다.
- [X] 상대방(컴퓨터)는 숫자를 생성한다.
  - [X] 1에서 9까지 서로 다른 임의의 수 3개를 선택한다.
- [X] 심판 객체 생성
  - [X] 심판 객체는 입력값과 상대방(컴퓨터)값을 비교 가능하다.
    - [X] 같은 수가 같은 자리에 있으면 '스트라이크', 다른 자리에 있으면 '볼', 같은 수가 전혀 없으면 '낫싱'이란 힌트를 준다.
    - [X] 게임 플레이어가 상대방(컴퓨터)의 3자리 수를 모두 맞히면 게임을 종료한다.
- [X] 숫자 야구 게임을 진행한다.
- [X] 숫자 야구 게임 결과를 출력한다.
