## 숫자야구게임 기능목록

- 1부터 9까지 서로 다른 수로 이루어진 3자리 수(이하 컴퓨터 수)를 뽑는다.
- 숫자를 검증한다.
  - 세자리가 아닌 경우 유효하지 않다.
  - 각 자리의 값에 숫자가 1부터 9까지의 숫자가 아닌 값이 포함된 경우(0 포함) 유효하지 않다.
  - 각 자리 숫자가 서로 같은 경우 유효하지 않다.
- 두 수를 비교하여 스트라이크와 볼 갯수를 리턴한다.
  - 두 수가 동일한 경우 3스트라이크, 정답이다.
  - 두 수를 비교하여 같은 수가 같은 자리에 있는 경우 스트라이크를 하나 늘린다.  
  - 두 수를 비교하여 같은 수가 서로 다른 자리에 있는 경우 볼을 하나 늘린다. 
  - 비교 결과 0스트라이크 0볼이라면 낫싱이다. 
- 게임 시작을 위해 컴퓨터 수를 정하고 사용자로부터 숫자(이하 사용자 수)를 입력받는다.
- 컴퓨터 수와 사용자 수가 같지 않은 경우, 스트라이크와 볼 갯수를 표시한 후 다시 입력을 받는다.
- 컴퓨터 수와 사용자 수가 일치하는 경우, 사용자의 입력값에 따라 게임을 다시 시작하거나 완전히 종료할 수 있다.
- 잘못된 사용자 수를 입력받아 IllegalArgumentException 발생 시 애플리케이션을 종료한다.
